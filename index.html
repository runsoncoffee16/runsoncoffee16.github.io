HTML Table references

Table tags

Tags Name Description
<table> Table The wrapper element for all HTML tables.
<thead> Table head The set of rows containing headers in a table.
<tbody> Table body The set of rows containing table data.
<tr> Table row The table row container.
<td> Table data The table data container in a table row.
<tfoot> Table foot The set of rows containing table footer.

Table attributes

Attribute Name Description
colspan column span Defines how many columns a td element should span. So content is spread over different columns.
rowspan row span Defines how many rows a td element should span. So content is spread over different rows.

CSS properties

Position

Value Description
static Standard value
relative Set its position from the site.
absolute Set its position from the parent element.
fixed Set its position fixed on the screen, often used for the header.
sticky Set its position always the same on the screen, even when scrolling down.
